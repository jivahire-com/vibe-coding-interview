Challenge Authoring Reference

This page is for recruiters and engineers who build new coding interview challenges for JivaHire Vibe. Use the Challenge Authoring Guide tab to learn how to structure a challenge repository — the required files (metadata, rubric, hidden tests, traps), naming conventions, and how candidate-safe content is separated from grader-only content. The Grading Overview tab explains how a candidate's submission is scored: automated test runs, trap detection, and the multi-stage LLM evaluation that produces the final score you see on the dashboard.
